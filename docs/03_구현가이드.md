AgentLoop — 구현 가이드

코딩 Agent가 참조할 구현 순서 및 상세 지침


1. 구현 순서
Step 1: Backend 기본 셋업

backend/ 폴더 구조 생성
requirements.txt 작성: fastapi, uvicorn, pyyaml, python-multipart
config.yaml 생성: docs_root 경로 설정
config.py: yaml 읽어서 설정 관리
main.py: FastAPI app + CORS 미들웨어 + 라우터 등록

Step 2: 프로젝트 API 구현

services/project_service.py

list_projects(): docs_root 스캔 → 숫자로 시작하는 폴더만 필터 → Project 리스트 반환
get_project(name): 특정 프로젝트 폴더의 index.md 파싱
init_project(num, title): 폴더 생성 + CLAUDE.md + 000_index.md 생성


routers/projects.py: 위 서비스를 REST API로 노출

Step 3: 문서 API 구현

services/index_service.py

parse_index(path): index.md → Document 리스트 + WorkLog 리스트
정규식으로 마크다운 테이블 파싱


services/document_service.py

list_documents(project_name): index 파싱 결과 반환
get_document_content(project_name, filename): raw 마크다운 반환


routers/documents.py: REST API 노출

Step 4: Frontend 기본 셋업

frontend/ Vite + React + TypeScript 프로젝트 생성
Tailwind CSS 설치 및 설정
React Router 설정
API client 래퍼 (src/api/client.ts)

Step 5: Dashboard 페이지

pages/Dashboard.tsx: 프로젝트 목록
components/ProjectCard.tsx: 프로젝트 카드 (폴더명, 문서수, 수정일)
components/InitProjectModal.tsx: 새 프로젝트 생성 모달

Step 6: Project 상세 페이지

pages/Project.tsx: 레이아웃 (사이드바 + 메인)
components/DocumentList.tsx: 대분류별 문서 목록 (사이드바)
components/MarkdownViewer.tsx: 마크다운 렌더링 (메인 영역)
components/WorkLog.tsx: 작업 로그 타임라인

import re

2.3 index.md 파싱 핵심 로직
python# 카테고리 헤더 패턴
CATEGORY_HEADER = re.compile(r'^### (\d)xx\s+(.+)$')

# 테이블 행 패턴 (| code | filename | summary | status |)
TABLE_ROW = re.compile(r'^\|\s*(\S+)\s*\|\s*(.+?)\s*\|\s*(.*?)\s*\|\s*(\S+)\s*\|$')

# 작업 로그 행 패턴
WORKLOG_ROW = re.compile(r'^\|\s*(\d{4}-\d{2}-\d{2})\s*\|\s*(.+?)\s*\|\s*(.*?)\s*\|$')
